BIFROST SERVICE

B!Frost is a Secure reverse gateway to link cloud services and in-House services. With B!frost, a service running in the cloud can get access and interact with a service running in an enterprise network.

The cloud service has first to connect and authenticate on a b!frost gateway running also in the cloud ; next the b!frost cloud gateway make the bridge to a b!frost enterprise gateway running in the enterprise network, through a secure channel ; finally the b!frost enterprise gateway creates the link to the in-house enterprise service.
