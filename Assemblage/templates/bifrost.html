Bienvenue sur la présentation de BIFROST

B!Frost est une passerelle inverse sécurisée permettant de relier les services cloud et les services internes. Avec B!frost, un service fonctionnant dans le cloud peut accéder et interagir avec un service fonctionnant dans un réseau d'entreprise. Le service cloud doit d'abord se connecter et s'authentifier sur une passerelle b!frost fonctionnant également dans le nuage. Ensuite la passerelle b!frost cloud fait le pont vers une passerelle b!frost enterprise fonctionnant dans le réseau de l'entreprise, via un canal sécurisé. Enfin, la passerelle d'entreprise b!frost crée le lien avec le service d'entreprise interne.
